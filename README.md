# install server dependencies
```sh
$ cd server
$ npm install express body-parser mysql nodemon cors
```

# run server
```sh
$ cd server
$ node index.js
```
# run server with nodemon (refresh server at every code change)
```sh
$ cd server
$ npm run devStart
```

# open server url
> http://localhost:3001/


# install axios on client
```sh
$ cd client
$ npm i axios
```
